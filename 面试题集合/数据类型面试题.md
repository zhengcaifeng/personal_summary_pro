3. 输出什么

```
 console.log(undefined==null)
 console.log(undefined===null)
 console.log(isNaN('100'))
 console.log(parseInt('12a')===12)
 console.log(parseInt('12a3')===123)
```

解析：
parseInt 是看到非数就停止处理

isNaN 执行的时候进行了隐式转换 Number('100')
手动实现

```
function isNaN(n){
let a=Number(n)+''  //因为NaN==NaN 返回false
if(a=='NaN'){
return true
}else{
return false
}
}

```

答案：输出 true false false true false
